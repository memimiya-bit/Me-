## 核心理念

**痛点**：企业平均35%决策基于错误数据，清洗工作占数据团队60%工时。

**目标**：建立智能数据治理体系，让错误数据在进入系统前就被识别和修正。

## 数据可信度评估

### 可信度评分公式

```formula
数据可信度 = (
  {来源系统权威性} × 0.3 + 
  {历史准确率} × 0.4 + 
  {更新及时性} × 0.2 + 
  {验证次数} × 0.1
) × 100
```

### 各维度评分标准

| 维度 | 权重 | 评分标准 | 数据来源 |
|-----|-----|---------|---------|
| 来源系统权威性 | 30% | ERP=90, CRM=85, 手工=60, 第三方=70 | 系统标识 |
| 历史准确率 | 40% | 近30天准确记录/总记录 | 数据校验 |
| 更新及时性 | 20% | 1天内=100, 3天内=80, 7天内=60, 7天外=30 | 时间戳 |
| 验证次数 | 10% | 每验证一次+10，最高100 | 人工确认 |

### 可信度等级

| 可信度 | 等级 | 处理方式 |
|-------|-----|---------|
| >=90 | A | 自动入库，无需审核 |
| 75-89 | B | 自动入库，标记待复核 |
| 60-74 | C | 需AI清洗后入库 |
| <60 | D | 驳回，人工处理 |

## 自动清洗规则库

### 规则类型

| 类型 | 规则数量 | 示例 |
|-----|---------|-----|
| 格式清洗 | 15+ | 日期格式、金额精度、文本规范 |
| 逻辑校验 | 20+ | 负数检查、未来日期、超限检查 |
| 重复检测 | 5+ | 完全重复、模糊重复、时间窗口去重 |
| 关联校验 | 10+ | 外键校验、业务规则校验 |
| 智能修正 | 8+ | 缺失值填充、异常值平滑 |

### 核心规则库

#### 格式清洗规则

```formula
// 金额格式标准化
清洗金额 = IF(
  REGEXMATCH({原始值}, "\\."),
  ROUND({原始值}, 2),
  {原始值}
)

// 日期格式统一
清洗日期 = SWITCH(TRUE(),
  ISDATE({原始值}), {原始值},
  REGEXMATCH({原始值}, "^\\d{8}$"), DATEVALUE({原始值}),
  REGEXMATCH({原始值}, "^\\d{4}-\\d{2}-\\d{2}$"), {原始值},
  "无效日期"
)

// 文本去空格+标准化
清洗文本 = TRIM(SUBSTITUTE({原始值}, " ", ""))
```

#### 逻辑校验规则

```formula
// 金额不能为负
异常标记 = IF({金额} < 0, "标记异常-负数", "正常")

// 日期不能在未来
异常标记 = IF({日期} > TODAY() + 365, "标记异常-未来日期", "正常")

// 百分比必须在0-100
异常标记 = IF(OR({百分比} < 0, {百分比} > 100), "标记异常-超限", "正常")

// 预算必须大于0
异常标记 = IF({预算} <= 0, "标记异常-无效预算", "正常")
```

#### 重复检测规则

```formula
// 完全重复检测
重复标记 = IF(
  COUNTIFS(
    {唯一标识字段}, {当前值},
    {日期字段}, {当前日期},
    {状态字段}, "有效"
  ) > 1,
  "完全重复",
  "正常"
)

// 模糊重复检测
相似度 = JARO_WINKLER({文本1}, {文本2})
重复标记 = IF(相似度 > 0.95, "疑似重复", "正常")
```

#### 智能修正规则

```formula
// 缺失值填充（基于同类均值）
填充值 = IF(
  ISBLANK({当前值}),
  AVERAGEIFS({历史值}, {分类字段}, {当前分类}, {时间字段}, "近30天"),
  {当前值}
)

// 异常值平滑（基于标准差）
平滑值 = IF(
  ABS({当前值} - {均值}) > 3 * {标准差},
  {均值},
  {当前值}
)

// 部门名称补全
补全部门 = IF(
  REGEXMATCH({部门}, "[x{4e00}-x{9fa5}]{1,3}$"),
  VLOOKUP({部门}, 部门全称映射表, 2),
  {部门}
)
```

## 治理流程设计

### 标准治理流程

```
┌─────────────────────────────────────────────────────────────────┐
│                        数据治理流程                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│    原始数据 ──┬─── 可信度评估 ───→ {可信度>=90%?} ───→ 是 ─→ 自动入库
│              │                                      │           │
│              │                                      ↓ 否       │
│              │                              ┌─────────────┐    │
│              │                              │ AI生成清洗建议│    │
│              │                              └─────────────┘    │
│              │                                      │           │
│              │                                      ↓           │
│              │                              ┌─────────────┐    │
│              └─── 规则引擎 ──→ 发现异常 ──→│ 责任人确认   │    │
│                                            └─────────────┘    │
│                                                  │           │
│                                                  ↓           │
│                                            ┌─────────────┐    │
│                                            │ 更新规则库  │    │
│                                            └─────────────┘    │
│                                                  │           │
│                                                  ↓           │
│                                                自动入库        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 流程节点说明

| 节点 | 触发条件 | 处理方式 | 时效要求 |
|-----|---------|---------|---------|
| 格式校验 | 数据入库 | 自动执行规则库 | 实时 |
| 逻辑校验 | 格式通过 | 执行业务规则 | 实时 |
| 可信度评估 | 校验通过 | 计算综合评分 | 实时 |
| AI清洗建议 | 可信度<90 | 生成修正方案 | <5秒 |
| 人工确认 | AI建议生成 | 责任人审核 | <4小时 |
| 规则更新 | 人工确认 | 更新规则库 | 实时 |

### 升级机制

```formula
升级条件 = AND(
  {异常类型} = "新类型",
  {发生次数} >= 5,
  {人工确认次数} >= 3
)

升级建议 = IF(升级条件,
  AI_GENERATE("为新异常类型'{异常描述}'生成清洗规则，包含判断条件和修正逻辑"),
  "无需升级"
)
```
