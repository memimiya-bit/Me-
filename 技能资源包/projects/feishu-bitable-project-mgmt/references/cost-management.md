## 战略层：项目群成本

### 项目群成本管理

**Portfolio成本汇总**：
```公式
项目群总预算 = SUM({项目预算})
项目群实际成本 = SUM({项目实际成本})
项目群预计收益 = SUM({项目预计收益})
项目群ROI = (项目群预计收益 - 项目群实际成本) / 项目群实际成本 × 100%
```

### 资源配置优化

**资源约束分析**：
```公式
资源约束指数 = 总资源需求 / 可用资源 × 100
资源瓶颈识别 = IF(资源约束指数 > 100, "存在瓶颈", "资源充足")
```

**优化建议生成**：
```公式
优化建议 = IF(资源约束指数 > 100,
  AI_GENERATE("资源约束指数为" & 约束指数 & "%，请提供资源配置优化建议，重点考虑：" & 高优先级项目 & "的资源保障"),
  "资源配置合理")
```

### 项目优先级排序

```公式
项目优先级 = 项目战略价值×0.4 + 财务回报×0.3 + 客户重要性×0.2 + 风险可控性×0.1

优先级排序 = RANK({项目优先级}, {所有项目优先级}, DESC)
```

## 战术层：项目成本

### EVM挣值分析

**基础指标计算**：

| 指标 | 公式 | 说明 |
|-----|------|------|
| 计划价值(PV) | =计划工作量×预算单价 | 计划应完成工作的预算 |
| 挣值(EV) | =完成工作量×预算单价 | 实际完成工作的价值 |
| 实际成本(AC) | =直接成本+间接成本 | 实际花费 |

**绩效指标**：
```公式
// 进度绩效指数 SPI
SPI = EV / PV
// >1 进度超前，<1 进度滞后

// 成本绩效指数 CPI
CPI = EV / AC
// >1 成本节约，<1 成本超支

// 进度偏差 SV
SV = EV - PV
// >0 进度超前，<0 进度滞后

// 成本偏差 CV
CV = EV - AC
// >0 成本节约，<0 成本超支
```

### 完工预测

**完工预算(EAC)**：
```公式
// 方法1：按当前效率预测
EAC_方法1 = BAC / CPI
// BAC = 项目总预算

// 方法2：按实际成本预测
EAC_方法2 = AC + (BAC - EV)

// 方法3：按当前进度预测
EAC_方法3 = AC + ((BAC - EV) / SPI)

// 推荐预测（加权平均）
EAC_推荐 = EAC_方法1×0.4 + EAC_方法2×0.3 + EAC_方法3×0.3
```

**完工尚需估算(ETC)**：
```公式
ETC = EAC_推荐 - AC
```

**完工偏差(VAC)**：
```公式
VAC = BAC - EAC_推荐
// >0 预计节约，<0 预计超支
```

### 成本偏差监控

**成本偏差率**：
```公式
成本偏差率 = (AC - BAC) / BAC
进度偏差率 = (EV - PV) / PV
```

**偏差趋势分析**：
```公式
成本趋势 = IF(本期偏差率 < 上期偏差率, "改善", 
  IF(本期偏差率 = 上期偏差率, "持平", "恶化"))
```

## 执行层：任务成本

### 工时成本核算

**任务工时成本**：
```公式
任务工时成本 = SUM({资源类型工时} × {资源单价})
人力成本 = 研发工时×人力单价 + 管理人员工时×管理单价
设备成本 = 设备占用时间 × 设备台时单价
```

### 资源成本分摊

```公式
// 按工时分摊
成本分摊比例 = 任务工时 / 项目总工时
分摊成本 = 项目总成本 × 成本分摊比例

// 按产值分摊
产值分摊比例 = 任务产值 / 项目总产值
分摊成本 = 项目总成本 × 产值分摊比例
```

### 任务成本分析

```公式
// 任务成本偏差
任务成本偏差 = 任务实际成本 - 任务预算成本
任务成本偏差率 = 任务成本偏差 / 任务预算成本

// 任务效率
任务效率 = 任务预算工时 / 任务实际工时
// >1 效率高于预期，<1 效率低于预期
```

## 成本预警机制

### 多级预警体系

**成本预警等级**：
```公式
成本预警等级 = SWITCH(TRUE(),
  成本偏差率 > 0.2, "🔴 严重超支",
  成本偏差率 > 0.15, "🟠 重大超支", 
  成本偏差率 > 0.1, "🟡 一般超支",
  成本偏差率 > 0.05, "🔵 轻微超支",
  "✅ 正常"
)

SPI预警 = SWITCH(TRUE(),
  SPI < 0.85, "🔴 严重滞后",
  SPI < 0.9, "🟠 重大滞后",
  SPI < 0.95, "🟡 轻微滞后",
  "✅ 正常"
)
```

### 预警自动升级

```公式
// 升级条件
预警升级条件 = AND(
  成本预警等级 IN ["🔴 严重超支", "🟠 重大超支"],
  持续天数 >= 3,
  负责人确认 = FALSE
)

// 升级通知
升级通知 = IF(预警升级条件,
  AI_GENERATE("项目" & 项目名称 & "成本" & 预警等级 & "已持续" & 持续天数 & "天，请确认处理方案"),
  "无需升级")
```

### 预警触发规则

| 预警级别 | 触发条件 | 通知对象 | 处理时限 |
|---------|---------|---------|---------|
| 🔴 严重 | 偏差率>20% | 高管+PMO+项目经理 | 24小时 |
| 🟠 重大 | 偏差率>15% | PMO+项目经理 | 48小时 |
| 🟡 一般 | 偏差率>10% | 项目经理 | 1周 |
| 🔵 轻微 | 偏差率>5% | 项目经理 | 月度评审 |

## 成本分析模板

### 成本分析仪表盘

**卡片配置**：

| 卡片 | 指标 | 公式 | 目标值 |
|-----|------|------|-------|
| 预算执行率 | 已执行/总预算 | (AC/BAC)×100% | <100% |
| 成本偏差 | 实际-预算 | AC-BAC | >0 |
| SPI | 进度绩效 | EV/PV | >1 |
| CPI | 成本绩效 | EV/AC | >1 |
| EAC预测 | 完工预算 | BAC/CPI | <BAC |
| VAC预测 | 完工偏差 | BAC-EAC | >0 |
