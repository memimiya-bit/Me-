## 公式优先级策略

### 优先级矩阵

| 优先级 | 方案 | 成本 | 适用场景 |
|-------|------|-----|---------| 
| P0 | 免费公式 | 0点 | 基础计算、筛选、统计 |
| P1 | 公式+AI组合 | 低 | 复杂分析需AI辅助 |
| P2 | 纯AI处理 | 高 | 无法用公式实现时 |

### 选择决策树

```
需要实现功能
    │
    ▼
能否用基础运算符(+,-,*,/)实现？
    │ Yes → 使用免费公式
    ▼ No
能否用COUNTIF/SUMIF等统计函数实现？
    │ Yes → 使用免费公式
    ▼ No
能否拆分为多个免费公式组合？
    │ Yes → 拆分后使用免费公式
    ▼ No
是否需要AI理解语义？
    │ Yes → 使用P1/P2方案
    │ No → 回退到简化版免费方案
```

## EVM挣值分析公式

### 核心指标计算

| 指标 | 公式 | 说明 |
|-----|------|------|
| 计划价值(PV) | 计划工作量 × 预算单价 | 计划应完成工作的价值 |
| 挣值(EV) | 完成工作量 × 预算单价 | 实际完成工作的价值 |
| 实际成本(AC) | 实际支出 | 实际花费的金额 |

### 绩效指数计算

```公式
// 进度绩效指数 SPI
SPI = EV / PV
// >1 进度超前，<1 进度滞后

// 成本绩效指数 CPI
CPI = EV / AC
// >1 成本节约，<1 成本超支

// 进度偏差 SV
SV = EV - PV
// >0 进度超前，<0 进度滞后

// 成本偏差 CV
CV = EV - AC
// >0 成本节约，<0 成本超支
```

### 完工预测

```公式
// 完工预算 EAC (按当前效率)
EAC = BAC / CPI
// BAC = 项目总预算

// 完工预算 EAC (按当前进度)
EAC = AC + (BAC - EV)

// 完工预算 EAC (按计划进度)
EAC = AC + ((BAC - EV) / SPI)

// 完工尚需估算 ETC
ETC = EAC - AC

// 完工偏差 VAC
VAC = BAC - EAC
// >0 预计节约，<0 预计超支
```

### 成本预警

```公式
成本偏差率 = (AC - BAC) / BAC

预警等级 = IF(成本偏差率>0.15, "🔴 严重超支",
  IF(成本偏差率>0.1, "🟡 一般超支",
    IF(成本偏差率>=0, "🟢 正常", "🟢 成本节约")))

SPI预警 = IF(SPI<0.9, "🔴 进度严重滞后",
  IF(SPI<0.95, "🟡 进度轻微滞后", "🟢 进度正常"))

综合预警 = IF(OR(SPI<0.9, CPI<0.9), "🔴 需要立即干预",
  IF(OR(SPI<0.95, CPI<0.95), "🟡 需要关注", "🟢 状态正常"))
```

### EVM仪表盘配置

| 卡片 | 指标 | 公式 | 目标值 |
|-----|------|------|-------|
| 整体进度 | SPI | EV/PV | >1 |
| 成本状态 | CPI | EV/AC | >1 |
| 预算执行 | 成本偏差率 | (AC-BAC)/BAC | <0 |
| 完工预测 | EAC | BAC/CPI | <BAC |

## 项目进度监控公式

### 项目完成率

**公式**：
```公式
完成率 = COUNTIF({任务状态}, "已完成") / COUNTA({任务ID}) × 100%
```

**应用场景**：仪表盘显示项目整体完成百分比

**示例**：
```
任务总数：20
已完成：12
完成率 = 12/20 × 100% = 60%
```

### 里程碑达成预警

**公式**：
```公式
里程碑状态 = IF(AND({计划完成日期}<TODAY(), {完成状态}≠"已完成"), "已延期", IF({完成状态}="已完成", "已达成", "进行中"))
```

### 任务超时计算

**公式**：
```公式
超时天数 = IF(AND({计划完成日期}<TODAY(), {任务状态}≠"已完成"), TODAY()-{计划完成日期}, 0)
```

### 进度偏差分析

**公式**：
```公式
进度偏差 = {实际完成度} - {计划完成度}
进度偏差率 = {进度偏差} / {计划完成度} × 100%
```

**判断逻辑**：
| 偏差率 | 状态 |
|-------|------|
| > 10% | 严重超前 |
| 0-10% | 正常超前 |
| = 0 | 正常 |
| 0-10% | 轻度滞后 |
| > 10% | 严重滞后 |

## 资源调度公式

### 资源冲突检测

```公式
// 资源负载率
资源负载 = COUNTIFS(分配表, @资源, 日期范围, "<="&TODAY()+7) / 最大负荷 × 100

// 冲突指数
冲突指数 = (已分配工时 / 可用工时) × 100

// 冲突等级
冲突等级 = IF(冲突指数<80, "🟢 充裕",
  IF(冲突指数<100, "🟡 紧张", "🔴 冲突"))
```

### 资源可用性判断

```公式
// 资源可用性
资源可用 = IF(已分配<可用容量, TRUE, FALSE)

// 最早可用时间
最早可用 = IF(资源可用, TODAY(), MIN(分配结束日期) + 1)
```

### 智能调度建议

```公式
调度建议 = IF(冲突指数>100, 
  AI_GENERATE("为["&@资源&"]在"&TEXT(TODAY()+7,"mm-dd")&"期间提供调度建议，考虑项目优先级和资源技能匹配"),
  "无需调整")

// 优先级排序
优先级得分 = 项目优先级×0.5 + 截止日期紧迫度×0.3 + 资源依赖度×0.2
```

### 资源优化公式

```公式
// 资源均衡指数
资源均衡 = 1 - (MAX(各周负载) - MIN(各周负载)) / AVERAGE(各周负载)

// 负载均衡优化
优化建议 = IF(资源均衡<0.7, "建议调整项目排期", "负载分布合理")
```

## 成本控制公式

### 项目成本偏差

**公式**：
```公式
成本偏差 = {实际支出} - {预算金额}
成本偏差率 = ({实际支出} - {预算金额}) / {预算金额} × 100%
```

**判断标准**：
| 偏差率 | 风险等级 |
|-------|---------|
| > 20% | 高风险 |
| 10-20% | 中风险 |
| 0-10% | 低风险 |
| < 0% | 节约 |

### 预算执行进度

**公式**：
```公式
预算执行率 = {已使用预算} / {总预算} × 100%
剩余预算 = {总预算} - {已使用预算}
```

### 人均产出计算

**公式**：
```公式
人均产出 = SUM({项目产出值}) / COUNTDISTINCT({项目成员})
```

### 成本效益分析

**公式**：
```公式
成本效益比 = {项目收益} / {项目成本}
投入产出比 = ({项目收益} - {项目成本}) / {项目成本} × 100%
```

### ROI分析

```公式
项目ROI = ({项目收益} - {项目成本}) / {项目成本} × 100%

ROI等级 = IF({项目ROI}>=50%, "优秀",
  IF({项目ROI}>=20%, "良好",
    IF({项目ROI}>=0%, "一般", "亏损")))

回收周期 = {项目成本} / ({项目收益}/12)
```

```公式
健康度等级 = IF(质量健康度>=90, "🟢 优秀",
```

### 缺陷加权评估

```公式
缺陷加权值 = COUNTIFS(严重度,"P0-致命")×10 + COUNTIFS(严重度,"P1-严重")×5 + COUNTIFS(严重度,"P2-一般")×2 + COUNTIFS(严重度,"P3-轻微")×1
```

### 缺陷趋势指标

```公式
// 缺陷收敛率
缺陷收敛率 = (本期新增缺陷数 - 本期关闭缺陷数) / 上期末遗留缺陷数 × 100%

// 缺陷逃逸率
缺陷逃逸率 = (生产环境缺陷数 / 总缺陷数) × 100%
```

```公式
缺陷预警 = IF(缺陷收敛率<0, "缺陷扩散",
  IF(缺陷逃逸率>10%, "逃逸严重",
```

**公式**：
```公式
项目周期 = {计划结束日期} - {计划开始日期}
```

**公式**：

## 公式组合示例

### 示例1：项目健康度综合评估

```公式
健康度得分 = 
  (IF({进度偏差}>=0, 40, 40-{进度偏差率}*10))
  + (IF({成本偏差率}<=0, 30, 30-{成本偏差率}*10))

健康度等级 = IF({健康度得分}≥90, "优秀", IF({健康度得分}≥70, "良好", IF({健康度得分}≥50, "一般", "较差")))
```

### 示例4：EVM完工预测

```公式
// 完工预算预测
BAC = 总预算
EV = 已完成工作量 * 预算单价
AC = 实际成本
CPI = EV / AC
SPI = EV / PV

// 三种预测方法
EAC_方法1 = BAC / CPI
EAC_方法2 = AC + (BAC - EV)
EAC_方法3 = AC + ((BAC - EV) / SPI)

// 推荐预测（加权平均）
EAC_推荐 = EAC_方法1 * 0.4 + EAC_方法2 * 0.3 + EAC_方法3 * 0.3

// 完工偏差
VAC = BAC - EAC_推荐
```
