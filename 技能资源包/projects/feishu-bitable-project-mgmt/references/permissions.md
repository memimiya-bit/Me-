## 权限体系概述

飞书多维表格的权限体系支持三个维度的控制：

1. **工作表级别**：控制对整张表的访问权限
2. **字段级别**：控制特定字段的可见性和可编辑性
3. **记录级别**：控制特定记录的访问权限

**本项目权限设计目标**：
- 确保数据安全，防止未授权访问
- 保障协作效率，避免权限过于宽松或严格
- 支持企业互联，安全共享必要数据给外部伙伴

### 角色一览

| 角色 | 适用人员 | 职责说明 |
|-----|---------|---------|
| 管理员 | IT负责人/项目经理 | 全面管理表格配置 |
| 项目经理 | PMO/项目总监 | 管理所有项目，查看全部数据 |
| 部门负责人 | 各部门经理/主管 | 管理本部门相关项目 |
| 普通成员 | 项目执行人员 | 仅管理自己负责的项目 |

## 企业互联配置

### 企业互联概述

企业互联是飞书提供的跨企业协作功能，允许：
- 邀请外部企业成员访问指定数据
- 设置外部成员的权限级别
- 隐藏敏感字段，保护商业机密
- 设置访问链接的有效期

### 启用企业互联

**操作路径**：
1. 打开多维表格
2. 点击右上角「分享」
3. 开启「企业互联」
4. 选择「允许外部企业访问」

### 创建外部访问链接

**步骤一：创建链接**
```
1. 在分享设置中点击「添加成员」
2. 选择「通过链接邀请」
3. 选择「外部企业成员」
4. 设置权限级别
5. 设置链接有效期（建议30天）
```

**步骤二：配置链接权限**
```
可访问范围：仅与我相关的记录
可见字段：排除敏感字段
可执行操作：仅查看和评论
```

**步骤三：分发链接**
```
生成链接后，通过邮件/短信发送给外部伙伴
提醒对方使用飞书账号登录
```

### 外部成员视图

## 敏感字段保护

### 敏感字段清单

| 字段类别 | 包含字段 | 保护级别 |
|---------|---------|---------|
| 财务数据 | 预算金额、实际花费 | 高 |
| 风险数据 | 风险等级、状态 | 中 |
| 组织数据 | 主责部门、负责人 | 中 |

### 隐藏配置方法

**方法一：通过字段权限设置**

1. 进入「高级设置」→「字段权限」
2. 选择「外部成员」角色
3. 将敏感字段设置为「不可见」

**方法二：通过视图筛选**

1. 创建「外部视图」
2. 配置筛选条件：「对接模式 = 企业对接」
3. 排除敏感字段的显示
4. 将此视图共享给外部成员

### 数据脱敏建议

对于必须显示的数据，可考虑脱敏处理：

| 原始数据 | 脱敏方式 | 示例 |
|---------|---------|------|
| 完整预算 | 范围区间 | 100-200万 |
| 详细花费 | 百分比 | 完成78% |
| 负责人姓名 | 仅显示部门 | 研发部 |
| 联系方式 | 部分隐藏 | 138****5678 |

## 权限配置步骤

### 步骤一：设置工作表基础权限

1. 打开多维表格
2. 点击「分享」→「工作表权限」
3. 设置默认权限为「仅协作者可编辑」
4. 添加管理员账号

### 步骤二：配置字段权限

1. 进入「高级设置」→「字段权限」
2. 按角色配置各字段的可见性
3. 特别注意敏感字段的保护

### 步骤三：设置视图权限

1. 创建不同角色的专属视图
2. 每个视图配置独立的筛选条件
3. 通过分享功能将视图分配给对应角色

### 步骤四：配置企业互联

1. 启用企业互联功能
2. 创建外部访问链接
3. 测试外部成员的访问效果

### 步骤五：权限审计

**定期检查项**：
- 新增成员是否正确分配角色
- 离职成员的访问权限是否及时撤销
- 敏感数据的访问记录是否正常
- 权限变更是否有日志记录
