## 概述

仪表盘是多维表格的数据可视化中心，通过卡片和图表展示项目整体状况，帮助管理者快速掌握项目健康度。

**仪表盘命名**：项目健康度总览

**访问路径**：
- 桌面端：多维表格 → 底部 Tab「仪表盘」
- 移动端：多维表格 App → 右下角「仪表盘」

## 卡片配置详解

### 卡片1：总项目数

**卡片类型**：统计卡片
**数据源**：项目对接全景表

**配置**：
```
统计方式：计数
字段：项目ID
分组维度：项目类型（可选）
显示格式：数字 + 小标题
```

**显示样式**：
```
        总项目数
         42
    ┌────┬────┬────┬────┐
    │ 12 │ 15 │ 10 │ 5  │
    └────┴────┴────┴────┘
  内部  供应 客户  合作
```

### 卡片2：延期项目占比

**卡片类型**：环形图
**数据源**：筛选「状态 = 已延期」

**配置**：
```
统计方式：计数
字段：项目ID
分组维度：无
图表类型：环形图
颜色：红色系
中心显示：延期数 / 总数
```

**显示样式**：
```
        延期占比
         7/42
        ┌─────┐
       │      │
       │  ◐   │
       │      │
        └─────┘
       16.7%
```

### 卡片3：预算使用率

**卡片类型**：进度条
**数据源**：汇总统计

**配置**：
```
计算方式：SUM(实际花费) / SUM(预算金额) * 100%
显示格式：百分比 + 数值
阈值设置：
  - 0-70%：绿色（正常）
  - 70-90%：黄色（警告）
  - 90%+：红色（危险）
```

**显示样式**：
```
      预算使用率
         78.5%
  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓░░░
  ¥785,000 / ¥1,000,000
```

### 卡片4：高风险项目列表

**卡片类型**：表格
**数据源**：筛选「风险等级 = 高风险 OR 风险等级 = 中风险」

**配置**：
```
显示条数：最多5条
排序：按风险等级降序
显示字段：
  - 项目名称
  - 风险等级
  - 延期天数
  - 完成度
  - 项目负责人
点击操作：跳转到对应记录
```

**表格样式**：
```
┌─────────────────┬────────┬────────┬────────┬──────────┐
│    项目名称     │风险等级│延期天数│ 完成度 │ 负责人   │
├─────────────────┼────────┼────────┼────────┼──────────┤
│ XX采购项目      │ 🔴高风险│  12天  │  60%   │ 张三     │
│ YY研发项目      │ 🟠中风险│   5天  │  45%   │ 李四     │
│ ZZ定制开发     │ 🟠中风险│   4天  │  80%   │ 王五     │
└─────────────────┴────────┴────────┴────────┴──────────┘
```

### 卡片5：月度完成趋势

**卡片类型**：折线图/面积图
**数据源**：按月份统计已完成项目数

**配置**：
```
X轴：月份（最近12个月）
Y轴：完成项目数量
图表类型：折线图（带面积填充）
趋势线：添加7日移动平均线
对比基准：显示去年同期数据（可选）
```

**显示样式**：
```
        月度完成趋势
  15 ┤
     │           ╱╲
  12 ┤      ╱───╱  ╲
     │  ╱───╱      ╲───╱
   8 ┤─╱                   ╲
     │╱
   0 ┼─────────────────────
     1月 2月 3月 4月 5月 6月
```

## 图表配置

### 常用图表类型选择

| 数据特征 | 推荐图表 | 说明 |
|---------|---------|------|
| 单一数值统计 | 统计卡片 | 简洁展示关键指标 |
| 占比分布 | 环形图/饼图 | 展示各类型占比 |
| 进度对比 | 进度条 | 展示完成度、预算使用率 |
| 时间趋势 | 折线图/面积图 | 展示月度变化趋势 |
| 排名对比 | 柱状图 | 展示部门/企业排名 |
| 多维度对比 | 雷达图 | 展示多指标综合对比 |

### 图表配色方案

**风险预警配色**：
```
高风险：#F53F3F（红色）
中风险：#FF7D00（橙色）
低风险：#FADB14（黄色）
无风险：#00B42A（绿色）
```

**项目类型配色**：
```
内部协作：#165DFF（蓝色）
供应商对接：#00B42A（绿色）
客户对接：#FF7D00（橙色）
合作伙伴：#722ED1（紫色）
```

## 视图关联

### 仪表盘与视图联动

仪表盘中的表格卡片支持点击跳转，直接打开对应的视图或记录。

**配置方法**：
1. 在卡片设置中开启「点击跳转」
2. 选择目标视图或记录详情页
3. 设置传递参数（如项目ID）

### 推荐关联视图

| 卡片 | 关联视图 | 传递参数 |
|-----|---------|---------|
| 高风险项目列表 | 风险预警视图 | 项目ID |
| 延期项目 | 企业对接看板 | 对接企业 |
| 预算超支项目 | 项目总览（按预算排序） | 无 |

### 仪表盘刷新设置

**刷新频率**：
- 实时数据：每次打开时刷新
- 分析数据：每小时自动刷新
- 历史数据：每日凌晨更新

**缓存策略**：
- 缓存时间：5分钟
- 后台更新：开启
