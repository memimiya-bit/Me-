## 流程概述

### 排产流程全景

```
需求整合 → 产能规划 → 生产执行 → 交付验收
    ↓          ↓          ↓          ↓
 产销协调会  产能评审会  每日站会   出货评审
```

### 服装企业排产特点

- 订单批量：通常为批量订单，按SKU生产
- 交期严格：服装季节性要求严格
- 产能波动：工人熟练度影响产能
- 工序复杂：多工序配合，瓶颈工序制约
- 质量要求：每个工序都需要质量检验

## 流程节点映射

### 节点一：需求整合

**触发时机**：销售订单确认
**关键动作**：
- 订单汇总
- 优先级排序
- 交期确认

**输出**：需求清单

### 节点二：产能规划

**触发时机**：需求清单确认
**关键动作**：
- 产能评估
- 工序分解
- 计划排布

**输出**：生产计划表

### 节点三：生产执行

**触发时机**：生产计划确认
**关键动作**：
- 物料发放
- 工序调度
- 进度跟踪

**输出**：生产日报

### 节点四：交付验收

**触发时机**：生产完成
**关键动作**：
- 成品检验
- 出货安排
- 客户验收

**输出**：出货单

### 订单状态选项

| 阶段 | 名称 | 说明 |
|-----|------|------|
| 1 | 待确认 | 订单待确认 |
| 2 | 已确认 | 订单已确认，待排产 |
| 3 | 已排产 | 生产计划已制定 |
| 4 | 生产中 | 正在生产 |
| 5 | 已完成 | 生产完成，待出货 |
| 6 | 已出货 | 已发货 |
| 7 | 已验收 | 客户已验收 |
| 8 | 已取消 | 订单取消 |

### 自动计算字段

```公式
待排产数量 = 订单数量 - 已排产数量
工序进度 = (已完成工序数 / 总工序数) * 100%
良品率 = (良品数 / 总生产数) * 100%
延期天数 = IF(实际结束日期 > 计划结束日期, 实际结束日期 - 计划结束日期, 0)
整体进度 = (良品数 / 订单数量) * 100%
```

### 风险预警规则

```公式
交期风险 = IF(AND(计划结束日期 - TODAY() < 5, 整体进度 < 80%), "高风险", "正常")
产能风险 = IF(实际产能 < 计划产能 * 0.8, "产能不足", "正常")
质量风险 = IF(良品率 < 95%, "质量预警", "正常")
延期风险 = IF(延期天数 > 0, "已延期", "正常")
综合风险 = IF(OR(交期风险="高风险", 质量风险="质量预警"), "需要关注", "正常跟进")
```

## 工作流配置

### 工作流一：自动排产

```
触发：当订单状态变更为"已确认"
动作：
  1. 计算交货期要求
  2. 检查产能可用性
  3. 自动分配产线和工序
  4. 生成生产计划
  5. 更新订单状态为"已排产"
```

### 工作流二：每日进度更新

```
触发：每天固定时间
动作：
  1. 汇总昨日生产数据
  2. 更新各订单进度
  3. 识别进度滞后订单
  4. 发送进度报告给项目经理
```

### 工作流三：交期预警

```
触发：计划结束日期前7天
条件：整体进度 < 90%
动作：
  1. 发送预警给生产经理
  2. 标记为"交期预警"
  3. 要求制定赶工方案
```

### 工作流四：延期处理

```
触发：计划结束日期到达
条件：整体进度 < 100%
动作：
  1. 自动更新延期天数
  2. 发送延期通知给客户
  3. 要求填写延期原因
  4. 如果延期超过3天：
     - 升级通知给销售总监
```

### 工作流五：质量预警

```
触发：当不良品数更新时
条件：良品率 < 90%
动作：
  1. 发送预警给质量部门
  2. 标记为"质量预警"
  3. 要求停工检查
  4. 记录质量问题
```

## 排产决策参考

### 优先级排序规则

| 优先级 | 条件 | 排序权重 |
|-------|------|---------|
| 紧急 | 交期 < 7天 | 100 |
| 高 | 交期 < 14天 或 VIP客户 | 80 |
| 中 | 普通订单 | 50 |
| 低 | 交期充足 | 20 |

### 产能分配规则

```规则
1. 优先分配紧急订单
2. 同优先级按交期先后排序
3. 考虑产线擅长品类
4. 避免频繁切换产品
5. 预留 10% 产能缓冲
```

### 延期处理策略

| 延期程度 | 处理方式 |
|---------|---------|
| 1-3天 | 与客户协商延期 |
| 3-7天 | 加急生产，部分空运 |
| > 7天 | 拆分订单，部分先发 |
