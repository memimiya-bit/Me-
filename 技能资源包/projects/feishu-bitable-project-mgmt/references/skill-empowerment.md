# 实用导向的岗位技能赋能体系

## 核心理念

**"技能即生产力，场景即战场"** —— 以岗位职能为中心，构建问题解决导向的技能赋能体系，让每个员工都能快速获得战场制胜能力。

---

## 一、实用导向核心逻辑重构

### 1.1 四层实用价值链

```
问题/需求 → 解决方案 → 技能工具包 → 实战成果
```

| 层级 | 内容 | 示例 |
|-----|------|------|
| **问题/需求** | 高频痛点、业务需求、个人挑战 | 跨部门协作效率低、项目交付提速30% |
| **解决方案** | 标准化流程、自动化工具 | 协作流程、进度追踪、优先级管理 |
| **技能工具包** | 飞书工具、知识库、智能提醒 | 多维表格+自动化、知识库+OKR |
| **实战成果** | 可量化成果 | 周期缩短25%、会议减少40% |

### 1.2 问题-技能精准匹配引擎

```formula
// 问题诊断表
问题诊断 = {
  "问题分类": "效率/质量/沟通/风险",
  "影响范围": "个人/团队/部门/公司",
  "紧急程度": "高/中/低",
  "复杂度": "简单/中等/复杂"
}

// 技能匹配度算法
技能匹配度 = (
  {问题相关度} * 0.4 + 
  {用户熟练度} * 0.3 + 
  {实施成本} * 0.2 + 
  {预期收益} * 0.1
) * 100
```

**匹配示例**：
- 用户问题："多个项目进度混乱，无法及时发现延期风险"
- 匹配技能：
  - 核心技能：依赖路径可视化_v2
  - 辅助技能：智能进度预警_v1、资源冲突检测_v1
  - 快速启动：3分钟创建项目依赖看板模板

### 5.2 认证评估标准

```formula
// 认证评分
认证评分 = (
  {理论考核} * 0.3 + 
  {实操评估} * 0.4 + 
  {场景测试} * 0.2 + 
  {同行评议} * 0.1
)

// 通过标准
通过 = AND(
  认证评分 >= 80,
  实操评估 >= 75,
  无重大失误记录
)
```
