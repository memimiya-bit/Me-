## 错误类型分类

### 分类总览

| 错误类别 | 说明 | 影响级别 |
|---------|------|---------|
| 数据验证错误 | 字段值不符合规则 | 中 |
| 流程配置错误 | 流程节点配置不当 | 高 |
| 权限错误 | 用户无访问权限 | 中 |
| 性能错误 | 数据量过大或超时 | 高 |
| AI服务错误 | AI服务不可用 | 中 |

## 验证错误处理

### 数据验证错误

| 错误类型 | 错误消息 | 处理方式 |
|---------|---------|---------|
| 缺少必填字段 | `缺少必填字段：{field_name}` | 高亮字段，阻止提交 |
| 日期范围无效 | `开始日期不能晚于结束日期` | 高亮日期字段 |
| 重复记录 | `{field_name} 已存在相同值` | 提示检查并修改 |
| 数值超范围 | `{field_name} 超出允许范围 (min-max)` | 高亮并提示范围 |
| 格式错误 | `{field_name} 格式不正确` | 显示正确格式示例 |

### 流程配置错误

| 错误类型 | 错误消息 | 处理方式 |
|---------|---------|---------|
| 节点依赖缺失 | `流程节点 {node} 依赖未配置` | 提示补充依赖 |
| 循环依赖 | `检测到流程循环依赖` | 阻止保存，要求修改 |
| 触发条件冲突 | `与现有规则 {rule} 冲突` | 提示冲突并建议解决 |

### 权限错误

| 错误类型 | 错误消息 | 处理方式 |
|---------|---------|---------|
| 无访问权限 | `您没有访问此数据的权限` | 提示联系管理员 |
| 无编辑权限 | `您没有编辑此字段的权限` | 禁用编辑控件 |
| 无删除权限 | `您没有删除此记录的权限` | 隐藏删除按钮 |

## 兜底机制

### AI 服务降级

**触发条件**：AI 服务响应超时或返回错误

**降级策略**：
1. **第一级降级**：使用预定义模板替代 AI 生成
2. **第二级降级**：提供手动配置入口
3. **第三级降级**：返回基础版本并记录问题

```降级响应示例
{
  "status": "degraded",
  "message": "AI服务暂时不可用，已为您生成基础版本",
  "fallback": "standard_template",
  "suggestions": [
    "基础版本已包含核心字段",
    "您可以手动添加需要的字段",
    "AI服务恢复后会自动提示优化"
  ]
}
```

### 超时处理

**触发条件**：处理时间超过 60 秒

**处理策略**：
```
1. 返回已处理的部分结果
2. 标记未完成的项目
3. 提供继续处理入口
4. 记录断点位置
```

```超时响应示例
{
  "status": "partial_complete",
  "processed": 6,
  "total": 8,
  "incomplete": ["sheet_7", "sheet_8"],
  "resume_token": "xxx",
  "message": "处理超时，已完成6个页签，请点击继续完成剩余部分"
}
```

### 数据量过大处理

**触发条件**：单页签数据超过 10,000 行

**处理策略**：
```
1. 分批次生成，每批 5000 行
2. 生成索引表便于查询
3. 提供数据导出选项
4. 建议分表管理
```

```过大数据响应示例
{
  "status": "batch_processed",
  "batches": 3,
  "batch_urls": [
    "download_part1.xlsx",
    "download_part2.xlsx",
    "download_part3.xlsx"
  ],
  "index_sheet": "索引表.xlsx",
  "message": "数据已分3批处理，请下载完整数据"
}
```

## 用户引导

### 帮助提示

| 场景 | 提示文案 |
|-----|--------|
| 字段配置 | `如何优化字段配置？→ 点击字段设置图标` |
| 添加流程 | `如何添加新流程？→ 选择'自定义流程'选项` |
| 共享表格 | `如何共享表格？→ 点击'协作设置'按钮` |
| 数据导入 | `如何导入数据？→ 使用'导入向导'` |
| 权限设置 | `如何设置权限？→ 进入'权限管理'` |

### 引导流程

```
用户遇到问题
    ↓
显示相关帮助提示
    ↓
用户可选择：
  - 查看详细帮助文档
  - 联系管理员
  - 提交问题反馈
```

### 常见问题解答

| 问题 | 解答 |
|-----|------|
| Q: 如何添加新字段？ | A: 点击字段标题旁的"+"按钮，选择字段类型 |
| Q: 如何设置下拉选项？ | A: 进入字段设置，选择"单选/多选"，添加选项 |
| Q: 如何配置自动计算？ | A: 选择"公式"字段类型，输入计算公式 |
| Q: 如何设置条件格式？ | A: 进入"高级设置"→"条件格式" |
| Q: 如何生成图表？ | A: 切换到"仪表盘"视图，点击"添加图表" |
| Q: 如何共享给外部人员？ | A: 点击"分享"→"企业互联"→"创建外部链接" |

## 错误日志

### 日志记录内容

```日志格式
{
  "timestamp": "2024-01-15 10:30:00",
  "error_type": "validation_error",
  "field": "计划结束日期",
  "user": "user_id",
  "action": "create_record",
  "detail": "开始日期晚于结束日期",
  "resolution": "用户修正后重新提交"
}
```

### 日志用途

| 用途 | 说明 |
|-----|------|
| 问题追踪 | 定位错误发生的原因 |
| 用户支持 | 帮助客服快速定位问题 |
| 产品优化 | 统计高频错误，改进产品 |
| 审计合规 | 记录数据变更历史 |

## 性能参数

| 参数 | 默认值 | 说明 |
|-----|-------|------|
| 单页签最大行数 | 10,000 | 超出时提示分表 |
| 单文件最大页签数 | 12 | 超出时提示拆分为多文件 |
| 生成超时时间 | 60 秒 | 超出时返回部分结果 |
| 批量处理大小 | 5,000 行/批 | 大数据量时分批处理 |
| 缓存策略 | LRU 30天 | 最近30天模板缓存 |
| 压缩级别 | medium | 文件压缩比 |
| 输出格式 | xlsx/csv | 支持的格式 |

## 监控告警

### 告警条件

| 指标 | 告警阈值 | 处理措施 |
|-----|---------|---------|
| AI 服务错误率 | > 5% | 自动降级 + 通知运维 |
| 处理超时率 | > 10% | 扩容处理能力 |
| 用户报错率 | > 3% | 检查功能问题 |
| 内存使用率 | > 80% | 清理缓存 |

### 告警通知

```
触发条件满足
    ↓
发送告警通知（邮件/飞书）
    ↓
记录告警事件
    ↓
自动执行预设处理措施
```
