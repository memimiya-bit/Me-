## 基础信息区

### 项目ID

**字段类型**：公式
**公式**：
```
CONCATENATE("PRJ-", TEXT(ROW()-1, "000"))
```
**示例输出**：`PRJ-001`、`PRJ-002`、`PRJ-015`

### 项目名称

**字段类型**：单行文本
**验证规则**：长度 2-50 字符
**示例**：
- 「新产品转产项目」
- 「2024年度供应商评估」
- 「客户需求定制开发」

### 项目类型

**字段类型**：单选
**选项**：
| 选项 | 颜色 | 适用场景 |
|-----|------|---------|
| 内部协作 | 蓝色 | 跨部门协作项目 |
| 供应商对接 | 绿色 | 与上游供应商的合作 |
| 客户对接 | 橙色 | 与下游客户的合作 |
| 合作伙伴 | 紫色 | 与中游合作伙伴 |

### 对接模式

**字段类型**：单选
**选项**：
- 部门对接：用于内部部门间的项目协作
- 企业对接：用于与外部企业的项目对接

## 部门与人员

### 主责部门

**字段类型**：单选
**选项建议**：
```
研发中心
生产中心
营销中心
采购部
财务部
品质部
人力资源部
信息技术部
```

### 协作部门

**字段类型**：多选
**选项**：同主责部门
**说明**：可选择多个需要配合的部门

### 项目负责人

**字段类型**：人员选择
**约束**：单选，必填
**权限说明**：默认为记录的编辑者

### 对接负责人

**字段类型**：人员选择
**约束**：多选
**说明**：
- 内部对接人：企业内部的负责人
- 外部对接人：通过企业互联邀请的外部人员

## 外部对接信息

### 对接企业

**字段类型**：单行文本
**格式验证**：无特殊要求
**填写示例**：
- 「深圳市XX供应链有限公司」
- 「广州XX机械制造厂」
- 「香港XX贸易行」

### 企业类型

**字段类型**：单选
**选项**：
| 选项 | 说明 |
|-----|------|
| 上游供应商 | 提供原材料或半成品的企业 |
| 中游合作伙伴 | 共同完成项目的合作企业 |
| 下游客户 | 采购产品或服务的企业 |
| 服务商 | 提供专业服务的企业 |

### 联系方式

**字段类型**：单行文本
**格式建议**：`电话: xxx-xxxx-xxxx | 邮箱: xxx@xxx.com`
**验证规则**：
- 电话：支持手机号和座机
- 邮箱：标准邮箱格式

## 时间与进度

### 计划开始日期

**字段类型**：日期
**格式**：YYYY-MM-DD
**默认值**：创建日期

### 计划结束日期

**字段类型**：日期
**格式**：YYYY-MM-DD
**验证**：必须大于等于计划开始日期

### 实际完成日期

**字段类型**：日期
**格式**：YYYY-MM-DD
**说明**：项目验收通过后填写

### 完成度

**字段类型**：数字
**范围**：0-100
**单位**：百分比
**格式**：进度条显示
**填写说明**：
| 完成度 | 含义 |
|-------|------|
| 0% | 项目刚创建或未开始 |
| 25% | 需求确认阶段完成 |
| 50% | 方案设计/开发完成 |
| 75% | 执行/测试完成 |
| 100% | 验收完成 |

### 项目阶段

**字段类型**：单选
**选项**：
| 阶段 | 说明 | 典型完成度 |
|-----|------|-----------|
| 需求确认 | 收集并确认项目需求 | 0-25% |
| 方案设计 | 制定项目方案和计划 | 25-50% |
| 执行中 | 按方案执行项目 | 50-75% |
| 验收 | 项目完成，等待验收 | 75-100% |
| 完成 | 项目通过验收，正式结束 | 100% |
| 暂停 | 项目暂时停止 | 不变 |

## 成本与风险

### 预算金额

**字段类型**：数字
**单位**：人民币（元）
**格式**：货币格式，带千位分隔符
**示例**：`¥1,000,000`

### 实际花费

**字段类型**：数字
**单位**：人民币（元）
**格式**：货币格式
**示例**：`¥850,000`

### 风险等级

**字段类型**：公式（自动计算）
**计算逻辑**：
```
延期天数 = MAX(0, TODAY() - 计划结束日期)
预算超支率 = IF(预算金额>0, (实际花费-预算金额)/预算金额*100, 0)

IF(延期天数>7 OR 预算超支率>20, "高风险",
  IF(延期天数>=3 OR 预算超支率>=10, "中风险",
    IF(延期天数>0 OR 预算超支率>0, "低风险",
      "无风险")))
```

### 状态

**字段类型**：公式（自动计算）
**计算逻辑**：
```
IF(完成度=100, "已完成",
  IF(AND(计划结束日期<TODAY(), 完成度<100), "已延期",
    IF(完成度>0, "进行中", "待启动")))
```

## 沟通管理

### 沟通频率

**字段类型**：单选
**选项**：
| 选项 | 说明 |
|-----|------|
| 每日 | 需要每天沟通跟进 |
| 每周 | 每周至少一次沟通 |
| 每两周 | 每两周沟通一次 |
| 每月 | 每月沟通一次 |
| 按需 | 根据实际情况沟通 |

### 沟通方式

**字段类型**：多选
**选项**：
- 飞书群
- 会议
- 邮件
- 电话
- 视频会议

### 最后沟通日期

**字段类型**：日期
**说明**：记录最近一次沟通的日期
**更新方式**：手动更新或通过自动化规则记录

### 下次沟通计划

**字段类型**：日期
**计算逻辑**：
```
IF(沟通频率="每日", TODAY(),
  IF(沟通频率="每周", TODAY()+7,
    IF(沟通频率="每两周", TODAY()+14,
      IF(沟通频率="每月", EDATE(TODAY(),1), ""))))
```

## 系统字段

### 创建时间

**字段类型**：自动时间戳
**触发**：记录创建时自动填充
**格式**：YYYY-MM-DD HH:mm

### 更新时间

**字段类型**：自动时间戳
**触发**：记录修改时自动更新
**格式**：YYYY-MM-DD HH:mm

### 附件

**字段类型**：文件
**说明**：上传项目相关的文档、图片等
**格式支持**：支持所有飞书支持的格式
