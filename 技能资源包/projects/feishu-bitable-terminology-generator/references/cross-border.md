## 同步原则

### 核心原则

**仅同步"不可协商"的基础术语**

| 原则 | 说明 |
|------|------|
| 不可协商 | 所有站点必须统一的规则 |
| 可协商 | 本地化运营决定的规则 |
| 禁止同步 | 文化差异大，无法统一的规则 |

### 同步范围

| 范围 | 示例 | 是否同步 |
|------|------|----------|
| 数值类 | 7天、48小时、满300减50 | 是 |
| 流程节点 | 已发货=物流揽收记录 | 是 |
| 禁用行为 | 刷单、虚假交易定义 | 是 |
| 文化特有名词 | 龙、福袋、御中元 | 否 |
| 本地化运营 | 节日促销、活动规则 | 否 |
| 语言习惯 | 无对应中文的场景 | 否 |

## 同步内容

### 数值类术语（必须同步）

| 中文定义 | 英文 | 日语 | 俄语 | 说明 |
|----------|------|------|------|------|
| 7天无理由 | 7-day return | 7日間返品可 | 7-дневный возврат | 时间单位必须统一 |
| 48小时发货 | 48h shipping | 48時間出荷 | 48-часовая доставка | 时效必须统一 |
| 满300减50 | Spend 300, save 50 | 300元以上で50円引 | Потрать 300, сэкономь 50 | 门槛必须统一 |
| 保价15天 | 15-day price protection | 15日間価格保護 | 15-дневная защита цены | 期限必须统一 |

### 流程节点术语（必须同步）

| 中文定义 | 说明 | 跨境影响 |
|----------|------|----------|
| 已发货 | 物流揽收记录生成 | 影响超时判定 |
| 已签收 | 物流系统确认收货 | 影响退货计时 |
| 已付款 | 支付系统确认到账 | 影响发货计时 |

### 禁用行为术语（必须同步）

| 中文定义 | 说明 | 违规后果 |
|----------|------|----------|
| 刷单 | 虚假交易提升排名 | 店铺扣分/封号 |
| 虚假宣传 | 夸大或误导性描述 | 商品下架/罚款 |
| 虚假评价 | 伪造用户好评 | 评价删除/账号处罚 |

### 不同步内容

| 类型 | 示例 | 不同步原因 |
|------|------|------------|
| 文化特有名词 | 龙、御中元、福袋 | 文化差异大 |
| 本地节日 | 双十一、黑五 | 仅本地运营 |
| 宗教/政治 | 部分宗教符号 | 风险高 |
| 俚语/流行语 | 电商黑话 | 无对应翻译 |

## 同步决策表

### 常见场景决策

| 场景 | 是否同步 | 原因 |
|------|----------|------|
| 中文"保价"规则变更 | 是 | 直接影响跨境履约 |
| 中文"7天"变更为"10天" | 是 | 影响所有站点退货政策 |
| 日文"御中元"节日术语 | 否 | 仅日本本地运营 |
| 俄语"Праздник"翻译 | 否 | 中文无直接对应 |
| 德语"Schadenfreude" | 否 | 与电商无关 |

### 同步触发条件

| 触发类型 | 说明 | 示例 |
|----------|------|------|
| 强制同步 | 影响跨境履约的规则变更 | 保价期限调整 |
| 建议同步 | 提升用户体验 | 发货时效优化 |
| 可选同步 | 本地化团队决定 | 文化特有名词 |

### 同步优先级

| 优先级 | 内容 | 同步时限 |
|--------|------|----------|
| P0 | 数值类、流程节点 | 即时同步 |
| P1 | 禁用行为 | 24小时内 |
| P2 | 建议同步项 | 1周内 |

## 文化敏感词

### 管理原则

**文化敏感词独立管理，不强行统一**

### 典型敏感词

| 词汇 | 中文含义 | 西方语境 | 处理方式 |
|------|----------|----------|----------|
| 龙 | 吉祥/力量/皇权 | 部分负面 | 独立管理+文化注释 |
| 8 | 发财 | 无特殊 | 无需处理 |
| 红色 | 喜庆 | 警告/危险 | 视场景标注 |
| 白色 | 纯洁 | 死亡 | 视场景标注 |
| 龟 | 长寿 | 贬义 | 谨慎使用 |

### 管理方式

```
文化敏感词库
├── 词汇
├── 中文定义
├── 西方语境注释
├── 使用建议
├── 风险等级（高/中/低）
└── 审核要求（必审/可选）
```

### 跨境页面处理

| 场景 | 处理方式 |
|------|----------|
| 商品命名含"龙" | 添加文化注释："Dragon (Chinese symbol of power and good fortune)" |
| 促销页用红色 | 评估目标市场习惯，考虑本地化配色 |
| 传统节日推广 | 提前评估文化敏感性，必要时调整设计 |

## 自动化配置

### 中文变更触发同步

```
中文词库变更
    ↓
IF(变更类型 IN ["数值类", "流程节点", "禁用行为"])
    ↓
自动扫描海外站点关联术语
    ↓
生成同步建议
    ↓
管理员确认
    ↓
同步至海外站点
```

### 海外反馈触发中文审核

```
海外客诉涉及术语歧义
    ↓
IF(非中文术语歧义)
    ↓
创建中文术语待审工单
    ↓
中文词库管理员审核
    ↓
确认是否需要更新中文定义
```

### 同步状态追踪

| 字段 | 说明 |
|------|------|
| 术语名称 | 被同步的术语 |
| 中文版本 | 中文定义版本号 |
| 英文版本 | 英文翻译版本号 |
| 日语版本 | 日语翻译版本号 |
| 同步时间 | 最后同步时间 |
| 同步状态 | 已同步/待同步/同步失败 |
