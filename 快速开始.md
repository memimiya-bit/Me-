## 🚀 快速部署流程

### 第一步：安装Node.js（如果还没安装）

1. 访问: https://nodejs.org/
2. 下载 **LTS版本** (推荐 22.x)
3. 双击安装程序，**勾选 Add to PATH**
4. 完成安装，**打开新的命令提示符窗口**
5. 验证安装：
   ```cmd
   node --version
   npm --version
   ```

### 第二步：安装项目依赖

```cmd
cd e:\统计数据系统\应用资源包
npm install
```

等待5-10分钟... ⏳

### 第三步：配置数据库（可选，开发环境可用默认）

编辑文件: `e:\统计数据系统\应用资源包\.env`

```env
# 使用SQLite（最简单，无需安装数据库）
SUDA_DATABASE_URL=file:./dev.db
```

### 第四步：启动应用！

```cmd
npm run dev
```

浏览器会自动打开，或手动访问：  
👉 前端: http://localhost:5173  
👉 后端: http://localhost:3000

## 🐛 遇到问题？

### 常见问题快速解决

#### 问题1：npm install失败
```cmd
# 清除缓存重试
cd e:\统计数据系统\应用资源包
npm cache clean --force
del package-lock.json
rmdir /s /q node_modules
npm install
```

#### 问题2：端口被占用
```cmd
# 查找并终止占用进程
netstat -ano | findstr ":5173"
taskkill /F /PID 进程ID
```

#### 问题3：Node.js未找到
- 重新安装Node.js，**确保勾选Add to PATH**
- **重启命令提示符窗口**
- 如还不行，重启电脑

更多详细问题解决，请查看: [部署指南.md](部署指南.md)
