## 🚀 快速启动指引

### 5步开始使用

1. **安装Node.js**
   ```cmd
   # 访问 https://nodejs.org/ 下载LTS版本
   # 安装时确保勾选 Add to PATH
   ```

2. **安装项目依赖**
   ```cmd
   cd e:\统计数据系统\应用资源包
   npm install
   ```

3. **配置数据库**
   ```env
   # 编辑 e:\统计数据系统\应用资源包\.env
   # 使用SQLite（最简单）
   SUDA_DATABASE_URL=file:./dev.db
   ```

4. **启动应用**
   ```cmd
   npm run dev
   ```

5. **访问应用**
   - 前端: http://localhost:5173
   - 后端: http://localhost:3000
   - API文档: http://localhost:3000/api

详细步骤请查阅: [快速开始.md](快速开始.md)
