## 🚀 部署步骤

### 第一步：安装Node.js

#### Windows安装方法

1. **下载Node.js**:
   - 访问: https://nodejs.org/
   - 下载 LTS版本 (推荐 22.x)
   - 选择 Windows Installer (.msi)

2. **安装Node.js**:
   - 双击下载的安装程序
   - 点击 "Next" 跟随安装向导
   - 选择默认安装路径
   - **重要**: 勾选 "Add to PATH" 选项
   - 完成安装

3. **验证安装**:
   ```cmd
   # 打开新的命令提示符或PowerShell窗口
   node --version
   # 应该显示: v22.x.x
   
   npm --version
   # 应该显示: 10.x.x
   ```

---

### 第二步：安装项目依赖

#### 1. 进入应用目录
```cmd
cd e:\统计数据系统\应用资源包
```

#### 2. 安装npm依赖
```cmd
npm install
```

预计安装时间：5-10分钟（取决于网络速度）

#### 3. 验证依赖安装
```cmd
# 检查 node_modules 目录是否存在
dir node_modules
```

---

### 第三步：配置数据库

#### 1. 编辑环境变量文件
打开文件: `e:\统计数据系统\应用资源包\.env`

#### 2. 配置数据库连接
```env
# Application
# Server Configuration (NestJS)
SUDA_DATABASE_URL=你的数据库连接字符串

# 例如（MySQL）:
# SUDA_DATABASE_URL=mysql://username:password@localhost:3306/database_name

# 例如（PostgreSQL）:
# SUDA_DATABASE_URL=postgresql://username:password@localhost:5432/database_name

# 例如（SQLite - 开发环境）:
# SUDA_DATABASE_URL=file:./dev.db
```

#### 3. 配置其他环境变量
```env
# Client Configuration

# Logger
LOG_REQUEST_BODY=true
LOG_RESPONSE_BODY=true
```

---

### 第四步：启动开发服务器

#### 方式一：完整开发模式（推荐）
```cmd
cd e:\统计数据系统\应用资源包
npm run dev
```

这将同时启动：
- NestJS后端服务
- Vite React前端服务
- 实时热重载功能

#### 方式二：分别启动服务
```cmd
# 终端1: 启动后端服务
cd e:\统计数据系统\应用资源包
npm run dev:server

# 终端2: 启动前端服务
cd e:\统计数据系统\应用资源包
npm run dev:client
```

## 🐛 故障排查

### 问题1：Node.js未安装
**症状**: `node --version` 命令失败  
**解决**: 按本指南第一步安装Node.js

### 问题2：npm install失败
**症状**: 依赖安装过程出错  
**解决**:
```cmd
# 清除npm缓存
npm cache clean --force

# 删除node_modules和package-lock.json
rmdir /s /q node_modules
del package-lock.json

# 重新安装
npm install
```

### 问题3：数据库连接失败
**症状**: 启动时显示数据库连接错误  
**解决**:
- 检查数据库服务是否启动
- 验证连接字符串配置
- 确认数据库用户权限

### 问题4：端口被占用
**症状**: 显示端口已被占用  
**解决**:
```cmd
# Windows查找占用端口的进程
netstat -ano | findstr ":5173"
netstat -ano | findstr ":3000"

# 终止进程（替换PID为实际进程ID）
taskkill /F /PID 进程ID
```

### 问题5：技能服务启动失败
**症状**: Python服务启动出错  
**解决**:
- 检查Python版本
- 确认所有依赖已安装
- 检查端口5004是否可用
