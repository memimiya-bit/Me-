# 浏览器插件自动化配置 (RPA工作流)

## 概述
基于RPA（机器人流程自动化）理念的浏览器插件自动化配置，实现数据采集、表单填写、页面监控等自动化任务。

## 支持的浏览器插件

### 1. 数据采集类插件

#### Web Scraper / Data Miner
- **功能**: 网页数据抓取、表格提取、列表采集
- **适用场景**: 
  - 每日/周/月数据采集
  - 竞品价格监控
  - 行业数据收集
- **配置示例**:
```json
{
  "plugin_name": "web_scraper",
  "data_sources": [
    {
      "url": "https://example.com/data",
      "schedule": "daily",
      "selectors": {
        "title": "h1.title",
        "price": ".price-tag",
        "date": ".date-field"
      },
      "output_format": "csv"
    }
  ]
}
```

#### Table Capture
- **功能**: 网页表格数据提取
- **适用场景**: 
  - 财务报表数据
  - 销售数据统计
  - 库存数据采集
- **配置示例**:
```json
{
  "plugin_name": "table_capture",
  "target_tables": [
    {
      "url": "https://finance.example.com/reports",
      "table_selector": "table.data-table",
      "schedule": "weekly",
      "output_destination": "飞书智能表格"
    }
  ]
}
```

### 2. 表单自动化类插件

#### Auto Form Filler
- **功能**: 自动填写表单、批量提交
- **适用场景**:
  - 批量数据录入
  - 自动化测试
  - 定期报告提交
- **配置示例**:
```json
{
  "plugin_name": "auto_form_filler",
  "forms": [
    {
      "url": "https://admin.example.com/data-entry",
      "schedule": "daily",
      "fields": {
        "date": "{{today}}",
        "sales_data": "{{from_feishu_table}}",
        "status": "completed"
      },
      "submit_button": "#submit-btn"
    }
  ]
}
```

#### Form Auto Fill
- **功能**: 智能表单识别与填写
- **适用场景**:
  - 重复性表单填写
  - 多平台数据同步
  - 自动化注册

### 3. 页面监控类插件

#### Page Monitor
- **功能**: 页面变化监控、内容更新通知
- **适用场景**:
  - 网站内容更新监控
  - 价格变动通知
  - 库存状态监控
- **配置示例**:
```json
{
  "plugin_name": "page_monitor",
  "monitors": [
    {
      "url": "https://shop.example.com/product",
      "check_interval": "hourly",
      "monitored_elements": [
        {
          "selector": ".price",
          "condition": "change",
          "notification": "飞书消息"
        }
      ]
    }
  ]
}
```

#### Visualping
- **功能**: 视觉变化监控、截图对比
- **适用场景**:
  - 网站改版监控
  - 广告位监控
  - 竞品页面监控

### 4. 工作流自动化类插件

#### iMacros
- **功能**: 宏录制、脚本执行
- **适用场景**:
  - 复杂操作流程自动化
  - 跨页面操作
  - 多步骤任务执行
- **配置示例**:
```json
{
  "plugin_name": "imacros",
  "macros": [
    {
      "name": "daily_data_collection",
      "steps": [
        {
          "action": "navigate",
          "url": "https://data.example.com"
        },
        {
          "action": "click",
          "selector": "#login-btn"
        },
        {
          "action": "type",
          "selector": "#username",
          "value": "{{username}}"
        },
        {
          "action": "type",
          "selector": "#password",
          "value": "{{password}}"
        },
        {
          "action": "click",
          "selector": "#submit"
        },
        {
          "action": "wait",
          "seconds": 3
        },
        {
          "action": "extract_table",
          "selector": "table.data-table",
          "output": "飞书智能表格"
        }
      ],
      "schedule": "daily"
    }
  ]
}
```

#### Puppeteer Recorder
- **功能**: 浏览器自动化脚本录制
- **适用场景**:
  - 测试自动化
  - 数据采集
  - 表单填写

### 5. 数据同步类插件

#### Webhook Integrator
- **功能**: Webhook触发、数据推送
- **适用场景**:
  - 实时数据同步
  - 事件触发通知
  - 第三方系统集成
- **配置示例**:
```json
{
  "plugin_name": "webhook_integrator",
  "webhooks": [
    {
      "url": "https://api.feishu.cn/hook/xxx",
      "trigger": "data_update",
      "data_source": "飞书智能表格",
      "transform": {
        "message": "数据已更新: {{record_count}}条记录",
        "link": "{{record_url}}"
      }
    }
  ]
}
```

#### API Connector
- **功能**: API调用、数据转换
- **适用场景**:
  - 系统间数据同步
  - 数据格式转换
  - 批量数据导入导出

## RPA工作流配置

### 工作流1: 每日数据采集与同步

```json
{
  "workflow_name": "daily_data_collection",
  "schedule": "0 9 * * *",
  "steps": [
    {
      "step_id": 1,
      "plugin": "web_scraper",
      "action": "extract_data",
      "config": {
        "url": "https://data.example.com/daily",
        "selectors": {
          "sales": ".sales-data",
          "inventory": ".inventory-data",
          "orders": ".orders-data"
        }
      }
    },
    {
      "step_id": 2,
      "plugin": "data_transformer",
      "action": "format_data",
      "config": {
        "format": "json",
        "mapping": {
          "sales_date": "{{today}}",
          "total_sales": "{{sum(sales)}}"
        }
      }
    },
    {
      "step_id": 3,
      "plugin": "feishu_api",
      "action": "update_bitable",
      "config": {
        "app_token": "xxx",
        "table_id": "xxx",
        "data": "{{transformed_data}}"
      }
    },
    {
      "step_id": 4,
      "plugin": "notification",
      "action": "send_message",
      "config": {
        "channel": "飞书群组",
        "message": "今日数据采集完成，共{{record_count}}条记录"
      }
    }
  ]
}
```

## 部署说明

#### Chrome/Edge
1. 访问Chrome Web Store
2. 搜索并安装所需插件
3. 配置插件设置
4. 导入工作流配置

#### Firefox
1. 访问Firefox Add-ons
2. 搜索并安装所需插件
3. 配置插件设置
4. 导入工作流配置

### 2. 配置本地服务

```bash
# 启动本地服务
cd e:\统计数据系统
python -m uvicorn src.main:app --host 0.0.0.0 --port 5004
```

### 3. 测试工作流

```bash

  -d '{"workflow_name": "daily_data_collection"}'

curl http://localhost:5004/api/workflow/status/daily_data_collection
```

## 维护与优化

### 定期维护任务

1. **每日**:
   - 检查工作流执行状态
   - 查看错误日志
   - 验证数据采集结果

2. **每周**:
   - 分析工作流性能
   - 优化数据采集策略
   - 更新监控阈值

3. **每月**:
   - 审查工作流配置
   - 更新插件版本
   - 备份配置文件

### 性能优化

1. **并行执行**: 使用多线程/异步执行独立任务
2. **缓存策略**: 缓存频繁访问的数据
3. **增量更新**: 只处理变化的数据
4. **错误重试**: 实现智能重试机制

## 安全注意事项

1. **敏感信息保护**:
   - 使用环境变量存储密钥
   - 定期轮换API密钥
   - 加密存储配置文件

2. **访问控制**:
   - 限制API访问权限
   - 实现身份验证
   - 记录访问日志

3. **数据安全**:
   - 加密传输数据
   - 验证数据完整性
   - 定期备份数据

## 故障排查

### 常见问题

1. **插件无法启动**:
   - 检查浏览器兼容性
   - 验证插件权限
   - 查看浏览器控制台错误

2. **工作流执行失败**:
   - 检查配置文件语法
   - 验证API连接
   - 查看执行日志

3. **数据采集错误**:
   - 验证页面选择器
   - 检查页面结构变化
   - 测试网络连接
